# Repetition isn't redundancy

saying "tools shape thought" in an essay about software and in an essay about education isn't repeating yourself. the context changes what the sentence means, what it connects to, what the reader does with it. on a graph this is obvious — the same node touched from different paths arrives differently each time. on a page it looks like you said the same thing twice.
