# Self-contained means redundant

Every essay has to set up its own context because the format assumes you might land there first. So the same idea gets introduced from scratch in every essay that touches it. The [graph](/prose/this-site-is-designed) is supposed to be the structure — but the essays duplicate that structure internally because they can't rely on it.
